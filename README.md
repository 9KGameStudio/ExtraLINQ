# ExtraLINQ #
ExtraLINQ provides additional extension methods for collections.

## Collection Extension Methods ##
  - CountsExactly
  - CountsMax
  - CountsMin
  - ElementAt
  - IsEmpty
  - IsNullOrEmpty
  - None
  - Random
  - Shuffle
  - Without

## Download ##
The easiest way to integrate ExtraLINQ into your .NET project is through the NuGet package *ExtraLINQ*.
